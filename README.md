# Libraries Needed

## Client Side

Within the Client directory, please first run

```bash
npm install
```

followed by :

### d3 library

```bash
npm install d3 --save
```

### framer-motion lbrary

```bash
npm install framer-motion
```

### Chart JS library

```bash
npm install --save chart.js react-chartjs-2
```

## Server

Within the Server directory, please first run

```bash
npm install
```

followed by :

### Cors library

```bash
npm install cors
```

### express library

```bash
npm install express
```

### mysql2 library

```bash
npm install mysql2
```
